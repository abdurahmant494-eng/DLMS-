Academic Project Overview

This Digital Library Management System (DLMS) is developed as part of the Software Engineering course (CoSc3061) at Borana University. The system aims to modernize library services, providing efficient management of academic resources for students, faculty, and library staff.

System Features

Book Management

Complete catalog system with search, filtering, and digital access

User Management

Role-based access for students, librarians, and administrators

Loan System

Automated borrowing, returns, renewals, and fine calculation

Incident Reporting

Track and manage library facility issues and disturbances

Live System Demo

Experience the fully functional system. The backend is powered by Node.js/Express with MySQL database, and the frontend uses modern HTML5, CSS3, and JavaScript.
